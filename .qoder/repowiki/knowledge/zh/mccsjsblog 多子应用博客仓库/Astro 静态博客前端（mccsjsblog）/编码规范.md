- 所有运行时数据（settings/menus/friends/twikooEnvId）均在 SSR 阶段通过 try/catch 包裹的 `fetch(API_URL + '/api/...')` 获取，失败时回退到模块级默认常量，保证离线可用。
- 页面脚本通过监听 `astro:page-load` 与 `swup:page-view` 双事件初始化，并用闭包或 DOM data-* 标记防止 swup 切换后重复绑定。
- Markdown 内容在服务端统一走 markdown-it 渲染，并在 renderer.rules.image / withFancybox 中对 `<img>` 标签做二次正则改写，使其自动获得 `data-fancybox="gallery"` 属性。
- CSS 主题色一律通过 CSS 自定义属性（如 `var(--card-bg)`、`rgb(var(--fg-muted))`）在 BaseLayout 中声明，组件侧只引用变量名，不写死颜色值。
- 类型定义集中放在 `src/types.ts`，各页面与组件通过相对路径 import 共享 Post/Category/Tag/SiteSettings/MenuItem 等接口，禁止在组件内部重新声明相同形状的类型。
