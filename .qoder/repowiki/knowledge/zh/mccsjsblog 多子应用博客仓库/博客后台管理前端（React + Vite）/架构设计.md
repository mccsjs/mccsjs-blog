模块为独立的前端应用，入口 `src/main.tsx` 通过 `QueryClientProvider` 注入 TanStack Query，`App.tsx` 使用 `react-router-dom` 的嵌套路由：`/login` 为登录页，其余页面统一由 `layouts/AdminLayout.tsx` 包裹，该布局负责侧边栏导航、移动端遮罩、未登录时重定向到 `/login`。

- 数据层：`lib/api.ts` 封装统一 `fetch` 调用（`credentials: 'include'`，自动拼接 `VITE_API_URL`），业务页面通过 `@tanstack/react-query` 的 `useQuery` 拉取数据；认证通过 `lib/auth.ts` 暴露的 `better-auth` 客户端及 `hooks/useAuth.ts` 组合 hook 完成 session 获取、登录/注册/登出。
- 组件层：`components/ui/` 提供 Button/Input/Label/Select/Textarea 等基础原子组件，`SplitMarkdownEditor.tsx` 作为富文本编辑器复用；各功能域以 `pages/*.tsx` 单文件组织（Dashboard/Posts/Categories/Tags/Comments/Menus/Friends/Settings/PostEditor/Login）。
- 样式体系：Tailwind CSS v4（`@tailwindcss/vite` 插件）+ 自定义 CSS 变量（`--bg`、`--accent` 等），图标统一来自 `@iconify/react` 的 Lucide 集合。
- 构建：`vite.config.ts` 配置 `@` 路径别名指向 `src`，`package.json` 脚本 `build` 先执行 `tsc -b` 再 `vite build`，产物输出至 `dist/`。
