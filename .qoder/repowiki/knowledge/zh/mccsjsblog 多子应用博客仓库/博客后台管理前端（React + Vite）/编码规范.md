- 所有页面组件放在 `src/pages/` 下，按功能域拆分为独立文件并通过 `App.tsx` 中的嵌套 `<Route>` 集中注册。
- HTTP 请求统一走 `lib/api.ts` 的 `api<T>()` 封装，禁止在页面中直接调用原生 fetch。
- 认证相关逻辑集中在 `hooks/useAuth.ts`，页面通过该 hook 获取 session/user 并调用 login/signup/logout，不直接操作 better-auth client。
- UI 元素优先从 `components/ui/` 引入原子组件（Button/Input/Label 等），避免在各页面重复实现基础控件。
- 图标统一通过 `@iconify/react` 的 `lucide:*` 命名空间引用，不在 JSX 中内联 SVG。
- 样式采用 Tailwind utility class 配合 CSS 变量（如 `var(--bg)`、`var(--accent)`）定义主题色，避免硬编码颜色值。
