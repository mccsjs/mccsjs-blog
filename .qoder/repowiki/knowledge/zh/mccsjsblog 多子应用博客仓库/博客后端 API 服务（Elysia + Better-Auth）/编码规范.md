- 所有写操作路由统一以 `{ auth: true }` 装饰器声明鉴权，并在 handler 内通过 `user` 上下文判断返回 401。
- 请求体一律经 Zod schema 解析（`postCreateSchema`、`commentCreateSchema`、`categorySchema`、`tagSchema`、`menuSchema`、`friendSchema` 等），校验失败走全局 VALIDATION 错误处理器返回 422。
- 对外响应中主动剔除敏感字段（如 posts 列表返回时 `content` 被解构丢弃），仅暴露前端所需子集。
- 管理员专属接口统一挂于 `/api/admin/*` 命名空间，公开接口则位于 `/api/*`，形成清晰的路径前缀隔离。
- 可选关联更新采用 Prisma connect/set/disconnect 语法（如 `tags: { set: tagIds.map(...) }`、`type: typeId ? { connect: { id } } : { disconnect: true }`）。
