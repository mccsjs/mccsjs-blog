- 脚本统一以 `#!/usr/bin/env node` shebang 开头并通过 `process.argv.slice(2)` 解析命令行参数，缺少必要参数时打印用法后 `process.exit(1)`。
- 备份文件命名采用 `<原文件名>_<YYYY-MM-DDTHH-mm-ss><扩展名>` 的时间戳格式，时间字符串通过 `toISOString().replace(/[:.]/g, '-')` 生成以保证文件系统兼容。
- 备份目标统一落在仓库根下的 `.backup/` 目录，并按原相对路径重建子目录结构，便于按文件维度回溯。
- 对外部可选依赖（如 chokidar）采用 try/catch require + 失败时自动 npm install 的懒加载策略，避免在无依赖环境直接崩溃。
