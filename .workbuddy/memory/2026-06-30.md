## Bug 修复
- **VditorEditor.tsx**：修复 React 19 Strict Mode 下 Vditor 异步初始化导致的 `Cannot read properties of undefined (reading 'currentMode/element')` 错误。根因是 Vditor 构造函数异步，`after()` 回调前内部 editor 为 undefined。将 vditorRef 赋值移入 after() 回调，加 readyRef 标志位，destroy 加 try-catch 防御。
