# mccsjs-blog

Astro 7 + ElysiaJS + Bun 全栈博客系统
`*100%由AI进行构建*`
## 技术栈

- **前端**：Astro 7 + Vue + TailwindCSS
- **管理端**：Vue 3 + Element Plus + md-editor-v3
- **后端**：Bun + ElysiaJS + PostgreSQL + Redis
- **部署**：EdgeOne（国内加速）+ 海外云服务器

## 本地开发

```bash
cd backend && bun run dev      # 后端 API (4000)
cd frontend && npm run dev    # 前端博客 (4321)
cd admin && npm run dev        # 管理后台 (5173)
```
